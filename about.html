解決人類歷史上最困難的決定

lightbulb 設計初衷

「今天午餐吃什麼？」這大概是上班族和學生每天最常問、也最難回答的問題。 AEUST RNG (Random Number Generator) 的誕生，就是為了將這個繁瑣的決策過程，轉化為一種充滿樂趣的隨機體驗。

我們相信，生活中的小驚喜往往來自於不確定性。透過演算法的隨機抽選，我們希望幫助使用者跳脫慣性思維，去嘗試那些平常可能忽略的美味選擇，讓每一餐都成為一次小小的探險。

palette 極簡美學

在這個資訊爆炸的時代，我們選擇了黑白極簡風格與毛玻璃擬態 (Glassmorphism) 設計。去除多餘的廣告與干擾，只留下最純粹的功能與視覺體驗。

網站支援即時主題切換功能，無論是深夜沉靜的「午夜深藍」，還是清晨活力的「極簡亮白」，都能隨著使用者的心情與環境自由變換，提供最舒適的視覺感受。

code 開發技術

本網站採用純粹的 HTML5、CSS3 與 JavaScript 構建，不依賴龐大的框架，確保了輕量化與極致的載入速度。

結合了 LocalStorage 技術，系統能夠記住您的偏好設定，讓每一次的訪問都如同老朋友般熟悉。地圖功能則整合了 OpenStreetMap 與 Leaflet 技術，在保護隱私的前提下提供便捷的導航服務。
